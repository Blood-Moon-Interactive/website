Global Indicator

November 2025 Blood Moon Interactive Global

Displays and updates a screen image indicator based on a global value or player health. Supports up to five different indicator images with customizable thresholds, perfect for creating dynamic HUD elements and visual feedback systems.

Understanding the Global Indicator Behavior

The Global Indicator behavior is an essential tool for creating dynamic, responsive user interfaces in GameGuru MAX. It allows you to display visual indicators on screen that automatically update based on player health or custom global variables, providing immediate visual feedback that keeps players informed about game state changes.

Key Features

  • Dual Monitoring Systems: Monitor either player health or custom user global variables
  • Multi-State Indicators: Support for up to five different indicator images with unique thresholds
  • Flexible Positioning: Fully customizable screen position (X and Y coordinates)
  • Adjustable Size: Control indicator scale from 1-100 for perfect sizing
  • Custom Images: Use your own indicator images for complete visual control
  • Real-Time Updates: Automatically updates as values change without player intervention
  • Aspect Ratio Handling: Automatically maintains correct image proportions
  • Always Active: Continuously monitors values for instant feedback

Monitoring Options

The Global Indicator offers two distinct monitoring modes, each perfect for different game design needs:

Health Monitoring

When set to monitor player health, the indicator automatically displays different images based on the player's current health percentage. This is perfect for health bars, status icons, or visual health warnings that respond to damage and healing.

Global Variable Monitoring

When monitoring a user global variable, the indicator tracks any custom game state you define. This opens up endless possibilities: track player score, inventory items, quest progress, environmental conditions, or any other numeric value in your game.

Indicator System Explained

The behavior supports up to five indicator states, each with its own image and threshold value. The system displays the appropriate image based on where the current value falls within your defined thresholds:

  • First Indicator: Displays when value is at or below the first threshold (default: 0)
  • Second Indicator: Displays when value is between first and second thresholds (default: 1-25)
  • Third Indicator: Displays when value is between second and third thresholds (default: 26-50)
  • Fourth Indicator: Displays when value is between third and fourth thresholds (default: 51-75)
  • Fifth Indicator: Displays when value is above the fourth threshold (default: 76-100)

Configuration Tips

  • Screen Position: Use X=0-2 for left alignment, X=50 for center, X=98-100 for right. Y=0-10 for top, Y=50 for middle, Y=90-100 for bottom
  • Screen Scale: Start with 50 for medium-sized indicators, adjust to 25-35 for smaller HUD elements or 65-80 for prominent displays
  • Threshold Values: Plan your thresholds to create meaningful visual transitions (e.g., 0, 25, 50, 75, 100 for five equal states)
  • Image Design: Create indicator images with clear visual progression (e.g., empty to full, dim to bright, red to green)
  • Global Variables: Ensure your monitored global variable is properly initialized and updated by other behaviors
  • Always Active: Remember to set the object to AlwaysActive for continuous monitoring

Implementation Best Practices

To create the most effective Global Indicator setup:

  1. Plan Your Visual States: Design indicator images that clearly communicate the current state to players
  2. Set Meaningful Thresholds: Choose threshold values that align with important gameplay milestones
  3. Position Strategically: Place indicators where they won't obstruct gameplay but remain easily visible
  4. Test Across Resolutions: Verify indicator positioning and sizing work on different screen resolutions
  5. Coordinate with Other Systems: Use Global Indicator alongside global_monitor, global_activator, and global_modifier for complex systems
  6. Create Visual Hierarchy: Use different scales to emphasize primary vs. secondary indicators

Advanced Techniques

For experienced developers, the Global Indicator offers several advanced techniques:

  • Multiple Indicators: Use multiple Global Indicator objects to track different values simultaneously
  • Combined Monitoring: Create indicator systems that show both health and custom stats on different screen areas
  • Progressive Systems: Design thresholds that create progressive difficulty or progression feedback
  • Visual Storytelling: Use indicator changes to tell a story (e.g., item condition, environmental danger, character state)
  • Integration with Game Mechanics: Link indicators to quest systems, inventory management, or character progression
  • Custom Image Sequences: Create image sequences that tell a visual story as values change
Pro Tip

The Global Indicator is most effective when used as part of a comprehensive feedback system. Combine it with sound effects, particle systems, and other visual cues to create rich, informative player feedback. Common Use Cases

The Global Indicator excels in various game scenarios:

  • Health Systems: Visual health bars, heart meters, or status icons that respond to damage
  • Resource Management: Display ammo count, energy levels, or inventory status
  • Progress Tracking: Show quest completion, level progress, or objective status
  • Environmental Feedback: Indicate temperature, toxicity, or atmospheric conditions
  • Status Effects: Display buffs, debuffs, or temporary condition indicators
  • Difficulty Scaling: Visual representation of dynamic difficulty adjustments

Related Behaviors

The Global Indicator works excellently with other global management behaviors:

  • Global Monitor: Modify global variables that the indicator can then display
  • Global Activator: Trigger events when indicator thresholds are reached
  • Global Modifier: Adjust global values that indicators track
  • Health Indicator: For dedicated health display systems
